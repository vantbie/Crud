# Sistema de Votación de Temas

Aplicación web Fullstack para proponer, votar y gestionar temas de discusión en tiempo real. Desarrollada con Node.js, Express y SQLite, implementando arquitectura MVC y actualizaciones dinámicas sin recarga (SPA-feel).

## Características Técnicas

- **Arquitectura MVC:** Separación clara entre Modelos (DB), Vistas (EJS) y Controladores.
- **Interacciones Asíncronas:** Uso de JavaScript Vanilla y Fetch API para operaciones CRUD sin recargar la página.
- **Backend Robusto:** API RESTful construida con Express.js.
- **Persistencia de Datos:** Uso de SQLite para almacenamiento ligero y eficiente.
- **Diseño UI:** Interfaz moderna basada en tarjetas con CSS personalizado.
